Política de Privacidade

Esta é uma política base para projetos criados a partir do Android App Starter. Ajuste este documento antes de publicar um app real.

Dados coletados

  • Dados de conta informados pelo usuário, como nome e email.
  • Conteúdo criado no aplicativo, conforme o domínio do projeto final.
  • Dados técnicos necessários para autenticação, funcionamento e segurança.

Uso dos dados

Os dados são usados para autenticação, operação do aplicativo, suporte e melhoria da experiência do usuário.
